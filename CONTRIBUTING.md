# Contributing Guidelines

## How to Contribute

### Volunteer Task Categories
1. **Research & Clinical Trials** — Monitor trials, extract data, identify new opportunities
2. **Outreach & Community Engagement** — Map clinics, develop materials, plan events
3. **Logistics & Coordination** — Communication, scheduling, travel, document sharing

### Getting Started
1. Fork this repository
2. Create a new branch for your task: `git checkout -b task/your-task-description`
3. Make your changes and commit with a clear message
4. Open a pull request referencing the relevant issue number

### Communication Channels
- **Slack**: #prosthetic-outreach (primary team channel)
- **GitHub Issues**: For task tracking and bug reports
- **GitHub PRs**: For all code/text contributions

### Meeting Cadence
- Weekly virtual standup (30 min) — check OUTREACH.md for time
- Monthly all-hands sync (1 hr) — rotates between cities or Columbus hub
- Quarterly in-person strategy meeting — Columbus, OH

### Code of Conduct
Be respectful and inclusive. This is a volunteering initiative focused on improving prosthetic access for underserved communities.
